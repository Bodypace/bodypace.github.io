    Development

    Bodypace is open-source, feel free to contribute or copy it

    Source code is on GitHub. Everything is under Apache License 2.0.

    I (Robert Dorna) do this project for learning purposes (fullstack development), but wish for something like this to exist. Therefore, anyone can support me by contributing to Bodypace or coping its features. I don't guarantee this project to be developed long term as I would like to join a company that needs software engineers in healthtech, and that could create a conflict of interests forcing me to stop developing Bodypace.

    Here are useful links for contributors: User interface design, Project Board, Online account server API.

    More stuff will be added later, e.g. mobile app development builds (created with Expo EAS).

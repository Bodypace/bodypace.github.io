Bodypace

We do not recommend installing current "early release" version.

This version release goal is to set up development workflow and start building the app incrementally from a stable base. This version is not designed to provide enough functionality to the users or to make it a viable alternative to already available healthcare, medical and fitness apps.

Bodypace app is currently at production track and being reviewed on Google Play Store.

Internal release and review found some accessibility issues but overall app is usable and without any functionality errors or warnings (known so far).

When the app becomes available in Play Store a link will be added here or this page URL will be changed to redirect to that app Play Store page.

For now, if someone wants to download and install the app, this link can be used (Android only).

To install downloaded .apk two things have to be done:

  • In Settings > Applications > Your web browser (or file explorer if you open .apk from there)
    - enable installing from unknown sources
  • In Play Store > User icon (top right corner) > Play Protect > Settings (cog icon in top right corner)
    - turn off application scanning

These settings are very useful and should be on so please turn them back on after installing this app.

Use it only for preview purposes and uninstall this app before installing the official one from the Play Store.
